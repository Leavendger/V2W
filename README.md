# 音频转文字

## 需求
面向产品经理的 ai 会议助手

## 功能点
- [ ] 自动断句

- [ ] 自动标点

- [ ] 自动标题

- [ ] 自动总结

- [ ] 关键词

- [ ] 待办事项提取

- [ ] 章节划分

- [ ] 时间轴跳转

- [ ] 搜索全文

- [ ] 导出 Word

- [ ] 导出 SRT

- [ ] 导出 Markdown

- [ ] 导出字幕

- [ ] 多语言翻译

- [ ] 实时录音

- [ ] 在线播放时点击文字跳转到对应时间

## 架构
如果按 MVP（最小可用产品）来规划，2～4 周内就可以先完成一个具备上传音频 → 自动转写 → 区分说话人 → AI 总结 → 导出文档能力的版本，再逐步扩展实时转写、协作等高级功能。



## 启动命令

  cd V2W
  source venv/bin/activate
  python app.py
  # → http://127.0.0.1:8080

  首次运行会自动下载 Whisper medium 模型（约 1.5GB），后续秒启动。
