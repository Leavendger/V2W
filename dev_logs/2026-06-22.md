## 关键决策记录

1. **faster-whisper 而非 openai-whisper**：同精度下速度快 4 倍，内存占用更低
2. **threading 而非 Celery**：单用户 MVP 无需分布式任务队列，简单线程足够
3. **SQLite 而非 PostgreSQL**：MVP 阶段零运维负担
4. **原生 JS 而非前端框架**：避免引入 Node.js 构建工具链，保持项目单纯
5. **MVP 不做说话人分离**：减少 Hugging Face token 配置步骤，降低门槛
